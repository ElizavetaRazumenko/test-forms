# Многошаговая форма заявки на займ

Проект реализован на `React + TypeScript + Vite`.

## Что реализовано

- 3 шага формы с переключением через `react-router-dom`.
- Формы на `react-hook-form`.
- Обязательная валидация всех полей через `zod` + `@hookform/resolvers`.
- Маска телефона формата `0XXX XXX XXX`.
- Загрузка опций места работы из DummyJSON (`/products/category-list`) с переиспользованием результата.
- Финальная отправка заявки в DummyJSON (`/products/add`) и показ модального окна с итогом:
  `Поздравляем, <Фамилия> <Имя>. Вам одобрена <сумма> на <срок> дней.`

## Почему добавлены библиотеки

- `react-router-dom` — нужен для маршрутизации между шагами формы по ТЗ.
- `react-hook-form` — нужен для управления формами по ТЗ.
- `zod` и `@hookform/resolvers` — выбраны для типобезопасной схемной валидации и единого источника правил.

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение будет доступно в браузере по адресу из вывода Vite (обычно `http://localhost:5173`).

## Проверка работоспособности

1. Откройте первый шаг и попробуйте нажать `Далее` без заполнения — должны появиться ошибки.
2. Заполните шаг 1 и перейдите на шаг 2.
3. На шаге 2 проверьте, что список мест работы загружается из API.
4. Нажмите `Назад` и убедитесь, что введённые значения сохранены.
5. Перейдите на шаг 3, выберите сумму и срок.
6. Нажмите `Подать заявку` — после успешного ответа появится модальное окно с персонализированным текстом.
